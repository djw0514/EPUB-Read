# EPUB 在线阅读器

一个功能丰富的在线 EPUB 电子书阅读器，支持文字和图片显示、字体调整、主题切换、双页模式等功能。网页版阅读链接 https://epub.space.minimaxi.com/

## 功能特性

### 📚 核心功能
- **EPUB 文件支持**：支持标准 EPUB 格式电子书
- **拖拽上传**：支持拖拽文件或点击上传
- **书架管理**：自动保存已导入的电子书，支持快速访问
- **阅读进度**：自动记录阅读位置，下次打开继续阅读

### 📖 阅读体验
- **翻页方式**：
  - 点击按钮翻页
  - 键盘方向键翻页
  - 鼠标滚轮翻页
- **目录导航**：快速跳转到指定章节
- **页码显示**：实时显示当前页码和总页数
- **进度条**：可视化显示阅读进度

### 🎨 个性化设置
- **字体调整**：
  - 字体大小：12px - 32px
  - 字体样式：默认、宋体、黑体、等宽、微软雅黑、仿宋
- **主题模式**：
  - 亮色模式（白底黑字）
  - 护眼模式（米黄底深棕字）
  - 深色模式（深灰底浅灰字）
  - 绿意模式（浅绿底深绿字）
  - 自定义颜色
- **显示模式**：
  - 单页模式
  - 双页模式
- **其他设置**：
  - 行距调整：1.0 - 3.0
  - 页边距调整：0 - 100px
  - 亮度调整：50% - 150%

## 使用方法

### 1. 打开应用
在浏览器中打开 `index.html` 文件。

### 2. 导入电子书
- **方式一**：点击"选择文件"按钮，选择 EPUB 文件
- **方式二**：直接拖拽 EPUB 文件到上传区域
- **方式三**：点击工具栏的导入按钮

### 3. 阅读电子书
- 使用工具栏按钮或键盘方向键翻页
- 点击目录按钮查看章节列表
- 点击设置按钮调整阅读参数

### 4. 管理书架
- 点击书架按钮查看已导入的电子书
- 点击"阅读"按钮打开电子书
- 点击"删除"按钮从书架移除

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一页 |
| → | 下一页 |
| Ctrl/Cmd + + | 增大字体 |
| Ctrl/Cmd + - | 减小字体 |
| 鼠标滚轮 | 翻页 |

## 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **EPUB 解析**：[Epub.js](https://github.com/futurepress/epub.js/)
- **文件压缩**：JSZip
- **数据存储**：
  - LocalStorage（设置和书架信息）
  - IndexedDB（电子书文件数据）
